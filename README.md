## Hello World Web App

A basic Node.js/Express web application with a modern UI.

### Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the server:
   ```bash
   npm start
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

### What's included:
- Express.js server (`server.js`)
- Modern HTML/CSS/JS frontend in `public/` directory
- Beautiful gradient design with animations
- Real-time clock display

### Stop the server:
Press `Ctrl+C` in the terminal
